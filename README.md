# 📊 Sales Data Analysis Project

## 📌 Overview
This project analyzes sales data from a corrugated box manufacturing business to extract meaningful insights.

## 🛠 Tools Used
- Python (Pandas)
- Google Sheets / Excel
- Data Visualization

## 📈 Key Insights
- Total Revenue: ₹510000
- Total Profit: ₹145000
- Top Product: Printed Box
- Highest Revenue Client: Escorts
